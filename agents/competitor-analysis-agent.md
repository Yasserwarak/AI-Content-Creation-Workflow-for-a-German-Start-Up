# Competitor Analysis Agent

Purpose

The Competitor Analysis Agent analyzes competitors in the bicycle and e-mobility market.

It identifies trends, content formats, and successful topics used by competitors.


Tasks

- analyze competitor social media content
- identify recurring themes
- detect successful video formats
- summarize market insights


Output

The agent produces structured insights that will be used by the Content Ideas Manager.

Example outputs

- trending topics
- popular hooks
- content formats


Model

gpt-5.4-pro


Reason for Model Choice

Competitor analysis requires deeper reasoning and pattern recognition across multiple competitors and content types.
